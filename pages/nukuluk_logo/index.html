An animated logo I worked on for @nukuluk_nukuluk. This was projected during a live show. The model was rendered using three.js and the processed with hydra. I set up a few different processing chains and then toggled them at diffrent intervals, sometimes changing the speed of the interval itself. The chains are mostly modulation or feedback effects. If you click 'View Live" above you will be taken to an instance of the code I used to generate the video where you can see the effects applied in real time.
